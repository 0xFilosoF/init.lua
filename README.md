## init.lua - Neovim Configuration 

### Key Features
- **Base Distribution**: [NvChad](https://nvchad.com/) v2.5
- **Plugin Manager**: [Lazy.nvim](https://github.com/folke/lazy.nvim) for efficient plugin management
- **LSP Support**: Comprehensive language server support for 10+ languages through [Mason](https://github.com/williamboman/mason.nvim)
- **Git Integration**: Advanced git workflows with Fugitive, Diffview, and Gitsigns
- **Terminal Integration**: Seamless tmux navigation and terminal management
- **AI Integration**: Windsurf (ex-Codeium)
- **Other**: Support daps, snacks, noice, leap, trouble, etc

- **Neovim** >= 0.11.0 (uses new LSP configuration API)
- **Git** for plugin management and git features
- (Optional) **npm** with global pkg: **nx**
- (Optional) **deno** for markdown
- (Optional) **codelldb** for debugging
- (Optional) [lazygit](https://github.com/jesseduffield/lazygit)
